## 💡 项目简介

在当前大模型开发中，让智能体拥有长期的记忆，并能自己写代码进化是极具挑战的。如果不加以限制，上下文极易爆仓，沙箱执行极易遭受恶意代码攻击，错误循环更是会导致无限的 API 开销风暴。

**Daming OS (大明天子 OS)** 就是为了终结这些乱象而诞生的。它脱胎于高度复杂的闭源企业级生产环境，将其最核心、最底层的**防爆仓记忆引擎**与**防黑客成长沙箱**完全抽离解耦，打造为了一个轻量、即插即用的 Python 开源包。

## 🔄 记忆与成长系统的闭环流转

Daming OS 中的记忆系统与成长系统并非孤立运行，而是通过事件总线与日志通道实现了深度的双向反馈与闭环流转。整体数据与代码流转流程一目了然：

<p align="center">
  <img src="assets/openclaw_flow.png" alt="Daming OS OpenClaw Flow" width="800">
</p>

### 1. 记忆系统核心模块
记忆系统负责智能体的高并发语义存取与防爆仓治理，提供高速、洁净的记忆外脑能力：
* **两级高并发语义缓存**：基于文件锁实现微秒级热数据响应，避免频繁读取数据库。
* **原子态会话记录机**：通过文件级原子替换与防撕裂写入，确保当前会话的存储安全无竞态。
* **多维搜索引擎**：深度整合密集向量与全文稀疏检索，实现基于语义相似度与文本匹配的多维极速定位。
* **物理排序融合引擎**：多路检索结果融合排序，并在返回前进行物理长度截断，严防单条长记忆撑爆上下文。
* **防爆仓压缩网关**：内置信息密度筛选算法，在额度超限时自适应精简历史，并将冗余物理固化旁路归档。

### 2. 成长系统核心模块
成长系统负责拦截运行报错并实现安全的运行时代码自修复，提供闭环的智能体自治能力：
* **异常行为捕获器**：实时监听运行报错日志，利用指数衰退滑动窗口计算成长值积分，积满触发能力演化。
* **多智能体演化辩论机制**：基于红方挑刺、蓝方修复、白方总结的博弈架构，自动提取高质量代码修复补丁与最佳实践。
* **静态安全审计与隔离沙箱**：在隔离沙箱内运行编译级安全检测与冒烟测试，封锁高危导入与反射，确保代码无毒。
* **人类在环授权中心**：生成审核卡片交由管理员确认，通过文件替换执行毫秒级原子部署，保留物理冷备支持一键回滚。

### 3. 两大系统的闭环交互机制
* **异常沉淀为记忆**：成长系统拦截智能体运行中的所有报错，转化为日志作为负反馈记忆沉淀至底层数据库。
* **召回历史教训**：当成长系统触发代码辩论演化时，记忆系统自动召回历史相似报错与避坑指南，避免重复犯错。
* **部署后刷新缓存**：成长系统完成代码部署后发布完成事件，记忆系统接收并清空缓存，使智能体下轮即刻应用新行为规范。

## 🌟 五大运转亮点

* **按需检索**：响应全局配置，仅在报错、遇瓶颈或求助时唤醒记忆库，极致优化上下文开销。
* **物理截断**：强制进行字符切片，过滤低价值长文，从物理层面上杜绝因单条记忆过长导致的直接爆仓。
* **记忆去嵌套**：在存入记忆库前自动清洗输入中的历史提示标签，避免智能体陷入循环记忆的逻辑死结。
* **静态安检门**：在隔离沙箱中对补丁进行编译级静态分析，强制拦截并封锁高危模块与反射，严防提权。
* **毫秒级部署**：采用文件级替换实现无感热插拔式自愈，实时保留备份并支持一键安全回滚。

## 🚀 一键极速安装

在您的终端中执行以下一键安装命令，即可直接从远程 GitHub 仓库拉取并配置包及其全部依赖，无需繁琐的手动克隆和配置：
```bash
pip install git+https://github.com/dylanma8232-art/Daming-OS.git
```

### 脚手架一键生成工作区
安装完成后，在您自己的智能体项目根目录下，运行命令行工具一键生成配置骨架：
```bash
daming-os init --dir ./my-agent-workspace
```
这将在目标目录下自动生成 `AGENTS.md`（指令规范）、`USER.md`（管理员授权配置）、`.env` 配置文件等。
